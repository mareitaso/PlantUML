## PlantUMLの書き方

### まず準備すること
最初に
@startUML
最後に
@endUML
を書かなければなりません
```
@startUML
//処理内容
//処理内容
//処理内容
@endUML
```
### 通常の処理
:と;ではさむと通常の処理を書けます
```
:処理内容;
```
### if文
if文を書くには以下のように書いていきます
```
if(条件)then(yesかNoなど)

else
endif
```
elseはなくても大丈夫ですが、"endif"は無いとエラーになります。  
then(yes)は書かなくてもエラーは出ませんが、どちらに分岐するかわかりにくいので書いたほうがいいでしょう。
